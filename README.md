- 총 개발 인원: 7명

  - 관리자 페이지 쿠폰 기능
  - 마이 페이지 쿠폰
  - 결제 시 쿠폰 적용
  - 메인 페이지 신상품 상품 노출
  - 마이 페이지 - 쿠폰 내역 기능
  - 마이 페이지 - 마이리뷰 보기 기능
